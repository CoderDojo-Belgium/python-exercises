Schrijf een programma dat een woord inleest. Als het woord $$n$$ letters heeft dan moet het programma $$n$$ herhalingen van het woord uitschrijven, elk op een afzonderlijke regel.

Invoer

Een woord van $$n$$ letters.

Uitvoer

$$n$$ herhalingen van het gegeven woord, elk op een afzonderlijke regel.
